¿Qué es Bingo Intelligence?

Definición

Bingo Intelligence es una plataforma de Business Intelligence que destaca por su facilidad de uso y la potencia de las aplicaciones analíticas generadas.

Componentes

Los componentes de la plataforma son:

  • Bingo Análisis. Es la herramienta principal que utiliza el usuario de la aplicación. Con Bingo Análisis, los usuarios pueden crear o consultar los informes, y los técnicos pueden definir los catálogos (metainformación de las bases de datos). Gracias a los catálogos, los usuarios pueden acceder a sus datos utilizando el lenguaje propio de su negocio, y no necesitan conocer la estructura de las bases de datos, ni el lenguaje SQL.
  • Bingo Excel. Es un complemento a Microsoft Excel que permite integrar las características de consulta y análisis de Bingo Intelligence con las hojas de cálculo Excel.
  • Servidor Bingo. Es el servicio web que atiende las peticiones de los usuarios y que realiza las consultas a las bases de datos y al repositorio de Bingo Intelligence. Los clientes (Bingo Análisis y Bingo Excel) nunca se conectan directamente a las bases de datos, y lo hacen siempre a través del servidor Bingo.
  • Repositorio Bingo. Es una base de datos Microsoft SQL Server que mantiene toda la información de la plataforma. Conviene aclarar que el repositorio sólo mantiene la estructura de las bases de datos y de los informes corporativos, mientras que los datos propios de la empresa (ventas, pedidos, producción...) se encuentran en una base de datos independiente (habitualmente en un datawarehouse o datamart con fines analíticos).
  • Administrador de Bingo Intelligence. Es la aplicación que utiliza el administrador del sistema para instalar, configurar y actualizar la plataforma. Con el administrador de Bingo Intelligence, se instala el servidor, se crea el repositorio y se publican los links de instalación de Bingo Análisis y Bingo Excel.

Tecnología

La plataforma Bingo Intelligence utiliza fundamentalmente tecnología de Microsoft:

  • El servidor Bingo es un servicio ASP.NET que funciona sobre el servidor web Internet Information Services, y utiliza la tecnología de Windows Communication Foundation (WCF) para asegurar la integridad y confidencialidad de las comunicaciones entre los clientes y el servidor.
  • Tanto el servidor como los clientes necesitan la versión 4 del .NET Framework de Microsoft.
  • La instalación de los clientes (Bingo Análisis y Bingo Excel) se realiza desde la propia intranet utilizando una tecnología denominada ClickOnce. Gracias a esta tecnología, la instalación resulta fácil y segura, y se facilita enormemente el trabajo de implantación y actualización de la plataforma entre todos los usuarios.
  • Bingo Análisis es una aplicación web con un frond-end de escritorio. En cierto modo, es un híbrido entre las aplicaciones desktop tradicionales y las aplicaciones web. Bingo Análisis comparte con las aplicaciones web la posibilidad de instalar y actualizar la aplicación desde la web, y las comunicaciones se realizan siempre utilizando protocolos web (http/https). Por otro lado, no deja de ser una aplicación Windows, y aprovecha la potencia de la máquina, y las características de Windows, para ofrecer una mejor experiencia de usuario.
  • Bingo Excel es un complemento de Excel que utiliza la tecnología Visual Studio Tools for Office (VSTO). Esta tecnología sustituye a las antiguas macros de Excel, y proporciona una máxima integración entre las aplicaciones y el propio Excel.
  • El repositorio es una base de datos Microsoft SQL Server.
  • Bingo Intelligence es una solución ROLAP que puede conectarse a cualquier base de datos relacional. Bingo no impone restricciones en la tecnología o la estructura de la base de datos que contiene la información corporativa, y puede conectarse indistintamente a cualquier motor de base de datos relacional (Oracle, MySQL, Teradata...).
