Requerimientos mínimos

Requerimientos mínimos del servidor

El servidor de Bingo Intelligence necesita:

  • Windows XP, o superior.
  • Servidor IIS (con ASP.NET 4, y WCF)
  • .NET Framework 4
  • Microsoft SQL Server 2005, o superior (puede estar en un servidor independiente, y puede ser la versión gratuita Express)
  • Conexión a internet (sólo para validar el número de licencia)

Los requerimientos mínimos de hardware son los mismos que necesitan los componentes anteriores, y Bingo Intelligence no impone ningún requisito adicional. De todos modos, se recomienda 2Gb de RAM, por lo menos.

Los requerimientos recomendables, sin embargo, dependerán de la carga de trabajo que vaya a soportar (número de usuarios concurrentes, volumen de datos, etc.). En general, se recomienda utilizar un sistema operativo diseñado para servidores (Windows Server 2008, o superior), aunque no es imprescindible. También se recomienda ejecutar Windows Update regularmente.

Conviene tener en cuenta que la máxima carga de trabajo no la soportará el servidor de Bingo Intelligence, ni el repositorio de Bingo, ni los clientes de Bingo Análisis o Bingo Excel. La máxima carga de trabajo la tendrá el servidor donde residan los datos corporativos, ya que Bingo Intelligence no “resuelve” las consultas directamente, sino que redirige las peticiones de los usuarios hacia el servidor de datos. Es importante, por lo tanto, que ese servidor esté correctamente dimensionado y que el modelo de datos soporte el tipo de consulta que generará el usuario.

Si la base de datos que se quiere explotar tiene un volumen de datos reducido, Bingo Intelligence puede acceder directamente a dicha base de datos, y el rendimiento debe ser suficiente. En cambio, si se dispone de una cantidad importante de información, o se disponen de fuentes heterogéneas que se quieren mezclar, o cuando el número de usuarios sea elevado, entonces se recomienda construir una base de datos específica para los procesos analíticos (a esta base de datos se la denomina, habitualmente, datawarehouse o datamart).

Requerimientos mínimos de los clientes

Los clientes de Bingo Intelligence necesitan:

  • Windows XP, o superior.
  • .NET Framework 4
  • Microsoft Excel 2003, o superior (sólo si se quiere utilizar Bingo Excel)
  • Conectividad con el servidor de Bingo

Los requerimientos mínimos de hardware son los mismos que necesitan los componentes anteriores, y Bingo Intelligence no impone ningún requisito adicional. De todos modos, se recomienda 2Gb de RAM, por lo menos.

Para obtener la máxima funcionalidad, y la capacidad de trabajar con un volumen de datos mayor, se recomienda Microsoft Excel 2007 o Microsoft Excel 2010 (en lugar del Excel 2003).

Al igual que en el servidor, se recomienda tener los ordenadores lo más actualizados que sea posible, y ejecutar regularmente Windows Update.
