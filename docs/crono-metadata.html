Capa semántica

Crono Metadata es la herramienta que utiliza el técnico para definir la capa semántica. La capa semántica es una pieza fundamental e indispensable de cualquier solución Business Intelligence. La capa semántica es el diccionario de indicadores y conceptos que dispone el usuario de negocio para generar sus consultas e informes.


La capa semántica incluye información sobre la propia estructura de los datos y por eso se denomina también “metadatos” o “catálogo”. La capa semántica traduce la complejidad de las tablas, campos y relaciones de la base de datos en un lenguaje conocido por el usuario.

Crear un catálogo con Crono Metadata

Crono Metadata comparte las características de facilidad de uso del resto de la plataforma. De este modo, la creación de un catálogo es algo sencillo y rápido. Para crear un catálogo de Crono solo es necesario conocer la estructura de la base de datos y disponer de conocimientos básicos del lenguaje SQL.


Los pasos para crear un catálogo son:

  1. Establecer la conexión con la base de datos
  2. Definir las “estrellas” o “copos de nieve” del modelo de datos, especificando las tablas y las relaciones entre ellas.
  3. Organizar la información que requiere el usuario en una estructura de carpetas. Para ello, simplemente se tienen que definir todos los indicadores y dimensiones.

Tras estos sencillos pasos, el catálogo ya se puede publicar y el usuario de negocio dispondrá de autonomía para generar sus propias consultas sin necesidad de conocer la estructura de la base de datos o el lenguaje SQL.
