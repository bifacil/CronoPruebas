Productos

La solución de Crono consta de dos productos principales. Crono Analysis es la herramienta que utilizan los usuarios de negocio para acceder a los datos de la empresa de una manera fácil y sin dependencia de IT. Crono ETL es la solución que utilizan los técnicos para preparar el data warehouse siguiendo buenas prácticas.

Solución para todo tipo de proyectos y empresas

Crono resulta útil para todo tipo de empresas y proyectos. Nuestros clientes son Administraciones Públicas y empresas de todos los sectores, desde grandes empresas multinacionales hasta comercios u organizaciones locales. Con Crono se puede hacer el seguimiento de los gastos de una pequeña agencia de viajes hasta controlar la producción y distribución de cientos de miles de SKU…

Crono Analysis revoluciona la manera de acceder a la información. Con Crono Analysis los usuarios pueden realizar estudios de mercado, analizar tendencias o detectar situaciones anómalas de una manera rápida, visual y fiable.

¿Qué es?

Crono Analysis es la herramienta que utiliza el usuario de negocio para consultar informes y cuadros de mando. Si tiene los permisos necesarios, el usuario también puede generar nuevas consultas e informes. Los informes son pantallas dinámicas y muy visuales que combinan tablas con todo tipo de gráficos. Crono permite navegar y visualizar la información desde distintos puntos de vista, de manera agregada o detallada. Gracias a estas avanzadas opciones de visualización, el analista obtiene un conocimiento superior de la información disponible y puede tomar decisiones basadas en datos y no solo en intuiciones.

La consulta y generación de informes se realiza siempre mediante drag&drop y de manera muy ágil y sencilla. Gracias a la capa semántica, el usuario no necesita conocimientos de programación ni ningún lenguaje extraño para construir comparativas, evoluciones, analizar tendencias, detectar anomalías u oportunidades…

Crono Excel

Crono Analysis incluye un complemento de Microsoft Excel 2007 (o superior) para extraer fácilmente la información de los sistemas corporativos y cargarlos directamente en la hoja de cálculo Excel. Los informes generados con Crono Excel se mantienen siempre actualizados y sincronizados con los datos de origen resolviendo así los habituales problemas de los informes Excel (datos desactualizados, incorrectos, o con criterios dispares entre distintos departamentos).

Desde el primer día, el usuario accede a los datos utilizando su propio lenguaje de negocio, sin necesidad de conocimientos técnicos y sin necesidad de aprender nuevas herramientas. Una vez obtenidos los datos con Crono Excel, el usuario dispone de todas las potentes y conocidas funcionalidades de Excel para preparar los informes del modo más conveniente.

Data Warehouse

Crono ETL es una herramienta especializada en la creación de los procesos de aprovisionamiento de data warehouses.

En organizaciones medianas y grandes se recomienda siempre la creación de bases de datos específicas para los procesos analíticos. Estas bases de datos analíticas se llaman “data warehouse” y están diseñadas para responder rápidamente las consultas masivas de los usuarios.

Los procesos ETL (Extracción, Transformación, y Carga) son los responsables de cargar diariamente el DWH a partir de los sistemas transaccionales de la empresa.

Facilidad de desarrollo y máximo rendimiento con Crono ETL

La parte más compleja de cualquier proyecto BI y donde se invierte más tiempo y más recursos es precisamente el diseño y construcción de estas bases de datos.

Crono ETL es la mejor solución para crear estos procesos de una manera rápida, segura, óptima y mantenible. Nuestro software se ha diseñado precisamente para facilitar la creación de estos procesos de carga y para asegurar unos elevados estándares de calidad durante el ciclo de vida del proyecto.

Crono ETL es un generador de código SQL optimizado para las necesidades de los proyectos DWH. De manera totalmente automática, Crono ETL genera el código más farragoso y complicado de los procesos de carga (estrategias de carga, logs, auditoría,…), por lo que el programador puede centrarse en lo verdaderamente importante y lo que genera valor para el proyecto: Diseñar un buen modelo de datos y aplicar correctamente las reglas de negocio definidas por el usuario.

Metodologías ágiles

Defendemos que las metodologías ágiles son la única opción razonable de realizar un proyecto de desarrollo hoy en día. Es necesario entregar resultados de manera muy rápida e incremental, y es necesario diseñar aplicaciones mantenibles y que crezcan y evolucionen del mismo modo que lo hace el Negocio.

Crono ETL tiene estas consideraciones en cuenta y ofrece muchas características que favorecen la correcta aplicación de las metodologías ágiles (seguimiento de patrones de desarrollo, paradigmas DRY y KISS, facilidades para el control del código fuente, extracción de metadata, automatización de tareas, y un largo etcétera).
